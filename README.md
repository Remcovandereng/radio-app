# 📻 Nederland FM Mini

Nederland FM Mini is een mobiele Progressive Web App (PWA) en desktop-app voor het luisteren naar online radiozenders uit Nederland en de rest van de wereld.

## ✨ Functies

### 🇳🇱 Nederlandse zenders
- NPO Radio 1
- NPO Radio 2
- NPO 3FM
- NPO Radio 4 Klassiek
- NPO Radio 5
- NPO FunX
- NPO Radio 2 Soul & Jazz
- Qmusic
- Qmusic Het Foute Uur
- Radio 538
- Radio 10
- Sky Radio
- Veronica
- Slam!
- KINK
- Sublime
- Joe
- BNR Nieuwsradio

### 🌍 Wereldwijde radiozenders
De app haalt automatisch landen en radiozenders op via Radio Browser.

### 🔍 Zoekfunctie
Zoeken op:
- zendernaam
- land
- genre/tags

### ❤️ Favorieten
Favorieten blijven lokaal opgeslagen op het apparaat.

### 📱 Mobiele ondersteuning
- iPhone (Safari)
- Android (Chrome)
- Tablets
